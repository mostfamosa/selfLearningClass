תהליך השעתוק והקומפילציה

תהליך השעתוק (Transcription)

מה זה? תהליך ביולוגי שבו מולקולת ה-DNA משמשת כתבנית ליצירת מולקולת RNA שליח (mRNA), אשר נושאת את המידע ליצירת חלבונים.

תווים שמרכיבים את ה-DNA וה-mRNA:

ה-DNA וה-mRNA מורכבים מתווים (בסיסים חנקניים), אך יש הבדל קטן בין ה-DNA ל-mRNA:

  • A (אדנין) - קיים ב-DNA וגם ב-mRNA.
  • T (תימין) - קיים רק ב-DNA.
  • U (אורציל) - מופיע רק ב-mRNA.
  • G (גואנין) - קיים ב-DNA וגם ב-mRNA.
  • C (ציטוזין) - קיים ב-DNA וגם ב-mRNA.

התאמה בין התווים:

במהלך תהליך השעתוק, נוצרת התאמה בין התווים של ה-DNA ל-mRNA באופן הבא:

  • A ב-DNA מתאים ל-U ב-mRNA.
  • T ב-DNA מתאים ל-A ב-mRNA.
  • G ב-DNA מתאים ל-C ב-mRNA.
  • C ב-DNA מתאים ל-G ב-mRNA.

דוגמה:

להלן דוגמה לתהליך השעתוק:

        DNA:  TACGGCAT
        mRNA: AUGCCGUA

תהליך הקומפילציה (תכנות)

מה זה? תהליך שבו קוד מקור (Source Code) מתורגם לקוד מכונה (Machine Code) שהמחשב יכול להבין ולהריץ.

מה מתקבל? קובץ הרצה (Executable File), כמו `.exe` או `.class`, שמאפשר למחשב לבצע את התוכנית.

איך זה קורה?

  • הקומפיילר בודק את הקוד לאיתור שגיאות סינטקטיות ולוגיות.
  • הקומפיילר מתרגם את הקוד לפורמט בינארי או ל-Bytecode.
  • נוצר קובץ הרצה (Executable) שמאפשר להריץ את התוכנית.

דוגמה:

קוד מקור (Java):

        public class HelloWorld {
            public static void main(String[] args) {
                System.out.println("Hello, World!");
            }
        }

אחרי קומפילציה (Bytecode):

        0: getstatic     #2 // Field java/lang/System.out:Ljava/io/PrintStream;
        3: ldc           #3 // String Hello, World!
        5: invokevirtual #4 // Method java/io/PrintStream.println:(Ljava/lang/String;)V
        8: return

קובץ הרצה שנוצר:

ב-Java, הקובץ שנוצר יהיה `HelloWorld.class`, אותו ניתן להריץ בעזרת java HelloWorld.

פעילות חווייתית: שעתוק בפעולה!

יש על השולחן 3 מיכלים שכתוב עליהם "שעתוק" וקוביה. הנה ההנחיות:

  1. זרוק את הקוביה.
  2. לפי המספר שיצא בקוביה, בחר שאלה מהמיכל המתאים:
    • 1-2: מיכל 1
    • 3-4: מיכל 2
    • 5-6: מיכל 3
  3. קרא את השאלה שבחרת. היא תכיל רצף DNA.
  4. בצע את תהליך השעתוק על פי מה שלמדת:
    • הזן את רצף ה-DNA בקלט הראשון.
    • תרגם את רצף ה-DNA לרצף ה-mRNA לפי הכללים.
    • מלא את רצף ה-mRNA בקלט השני.

תרגול שעתוק

הזן את רצף ה- DNA וגם רצף ה- mRNA ולחץ על "לבדוק" כדי לראות אם תהליך השעתוק בוצע נכון.
