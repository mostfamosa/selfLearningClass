תהליך תרגום והרצה

תהליך התרגום (Translation)

מה זה? תהליך שבו ה-mRNA משמש כתבנית ליצירת חלבון. כל 3 בסיסים (קודון) ב-mRNA מתורגמים לחומצת אמינו.

שלבים בתהליך:

  • הריבוזום קורא את ה-mRNA.
  • הריבוזום מחבר את חומצות האמינו ליצירת חלבון.

הסבר על קודון התחלה וסיום:

קודון התחלה: הקודון הראשון ב-mRNA שמסמן את תחילת התרגום הוא AUG, המקודד לחומצת האמינו מטיונין (Met).
קודון סיום: הקודונים UAA, UAG, UGA מסמנים את סוף התרגום ואינם מקודדים לחומצות אמינו.

דוגמה:

(התרגום מתחיל מקודון AUG ונעצר בקודון UAA)

    mRNA: GCU AUG CCG UAC UAA
    Protein: Met - Pro - Tyr

זכרו: כל 3 בסיסים = חומצת אמינו אחת!

תהליך ההרצה (Execution)

מה זה? תהליך שבו קובץ הרצה (Executable) שמתקבל אחרי הקומפילציה מופעל, והמחשב מבצע את ההוראות שבקובץ.

שלבים בהרצה:

  • המחשב טוען את קובץ ההרצה לזיכרון.
  • המעבד מבצע את ההוראות בקובץ לפי סדר.
  • התוצאה מוצגת למשתמש או מתבצעת פעולה מסוימת.

דוגמה להרצה:

תהליך של הרצת תוכנית:

קוד מקור:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

קובץ הרצה:

לאחר הקומפילציה, קובץ בשם HelloWorld.class נוצר. כדי להריץ את התוכנית:

> java HelloWorld
Output: Hello, World!
